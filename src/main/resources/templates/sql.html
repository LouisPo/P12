LES LANGUAGES DES BASES DE DONNEE

Voyons maintenant comment modifier notre base grâce à leur propre langage

Car oui, comme nos autres applications, les bdd ont aussi leur propre manière de se faire comprendre, il existe encore une fois de nombreux languages mais les bdd ont cet avantage d'avoir un langage universel qui se spécialise ensuite en fonction de la base avec laquelle vous voulez communiquer. Ce langage universel que vous verrez se nomme le SQL pour Structured Query Language mais on peut quand même citez quelques variantes telles que le PostgreSQL ou le MySQL. Il est maintenant temps d'un petit exemple, imaginons que nous voulons une base se nommant personne possédant deux tables, c'est-à-dire 2 zones de stockages pour différentes informations.
Ici, la commande CREATE permet de créer de nouvelles structures que ce soit des bases ou des tables, à savoir que les commandes sont toujours écrites en majuscule. Nous avons désormais un endroit où stocker des données donc autant l'utiliser voyons comment faire.
Nous utilisons ici la commande insert pour entrer les informations de notre personne, en précisant bien où, et en rajoutant les '' indiquant que ce que nous rentrons est du texte.
Ce n'est ici qu'une des nombreuses possibilités du sql et des bdd, nous vous invitons évidemment à plus vous renseigner si vous êtes intéressé ou juste par curiosité.
